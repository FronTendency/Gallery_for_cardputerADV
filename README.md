## 🚀 How to Use

1. **Prepare your SD Card:**
   - Create a folder named `Pictures` in the root directory.
   - Copy your images (`.jpg`, `.png`) into this folder.

2. **Directory Structure should look like this:**
   ```text
   SD Card Root
   └── 📁 Pictures
       ├── photo1.jpg
       └── photo2.png
